01 ¿Qué es una dirección IP?

Una dirección IP funciona como la dirección postal de cada dispositivo en una red. Así como cada casa tiene una dirección única, cada computadora, teléfono o router tiene una IP que permite identificarlo y enviarle datos a través de internet o una red local.

// Tu PC en casa
192.168.1.10
Red local privada
// Google.com
142.250.80.46
IP pública global
// Tu router
192.168.1.1
Gateway de la red
// Loopback
127.0.0.1
Siempre: tú mismo
02 Estructura de IPv4 — 32 bits interactivos

Una dirección IPv4 tiene 32 bits divididos en 4 grupos de 8 bits llamados octetos. Cada octeto se representa en decimal (0–255). Hacé clic en cualquier bit para cambiar su valor y ver cómo cambia la dirección.

– ← resultado en decimal
→ Cambiá los bits para detectar automáticamente la clase de la IP
Cómo leer el primer octeto (ejemplo: 192)
Bits: 1·128 + 1·64 + 0·32 + 0·16 + 0·8 + 0·4 + 0·2 + 0·1
Cálculo: 128 + 64 = 192
Posiciones: 2⁷=128 · 2⁶=64 · 2⁵=32 · 2⁴=16 · 2³=8 · 2²=4 · 2¹=2 · 2⁰=1

Ejemplo de cálculo — AND bit a bit para obtener la red
IP: 192.168.10.55
= binario: 11000000.10101000.00001010.00110111
Máscara /24: 255.255.255.0

AND bit a bit: 11000000.10101000.00001010.00000000
Red resultante: 192.168.10.0/24
Fórmula de hosts: 2bits de host − 2 = 28 − 2 = 254 hosts (se restan la red y el broadcast)
05 IPv6 — El futuro del direccionamiento

IPv4 permite ~4.300 millones de direcciones. Con el crecimiento exponencial de dispositivos conectados, se agotaron. IPv6 usa 128 bits y ofrece 3,4 × 10³⁸ direcciones — suficientes para cada átomo de la Tierra.

El :: solo puede aparecer una vez en la dirección.
